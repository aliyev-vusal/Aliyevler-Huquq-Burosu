# EduPath Bakı — Xaricdə Təhsil Konsaltinqi

EduPath Bakı xarici universitetlərə qəbul, imtahan hazırlığı və viza dəstəyi sahəsində peşəkar konsaltinq xidmətləri təqdim edən veb saytdır.

## Xidmətlər

- Xarici universitetlərə qəbul konsaltinqi
- IELTS, SAT, YÖS imtahan hazırlığı
- Viza müraciəti dəstəyi
- Təqaüd proqramları üzrə məsləhət

## Texnologiyalar

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Cormorant Garamond, Manrope)

## İstifadə

Saytı lokal olaraq işə salmaq üçün `index.html` faylını brauzerdə açın.
